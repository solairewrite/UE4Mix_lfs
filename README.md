## Git
+ 只拉取最近一次的提交  
`git pull --depth=1 --allow-unrelated-histories`  
+ 远端仓库回滚
```
git reset bb28e5c3c02d64d26e88b3969466139a76755cf0
git push -f
```
+ 合并另一个版本  
`git merge bb28e5c3c02d64d26e88b3969466139a76755cf0 --allow-unrelated-histories`
